# Hangman Game (Python Source Code)

This is a beginner-friendly Hangman game written in Python. It’s fully functional and perfect for practicing:

✅ Loops  
✅ Conditionals  
✅ Lists & Strings  
✅ User input and logic building

## How to Run:

1. Make sure Python is installed.
2. Run the file:
   python hangman.py
3. Start guessing letters and try to solve the word before your attempts run out!
